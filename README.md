# 📦 Sistema de Controle de Estoque

## 📌 Sobre o Projeto

O **Sistema de Controle de Estoque** é uma aplicação desktop desenvolvida com o objetivo de auxiliar no gerenciamento de produtos, fornecedores e movimentações de estoque de uma empresa.

O sistema permite realizar cadastros, consultas e registros de entradas e saídas de produtos, mantendo as informações organizadas em um banco de dados.

O projeto foi desenvolvido utilizando **Java**, com interface gráfica em **Java Swing**, comunicação com banco de dados **MySQL** através de **JDBC** e organização do código utilizando o padrão **DAO (Data Access Object)**.

---

## 🎯 Objetivo

O principal objetivo do sistema é proporcionar uma maneira simples e eficiente de controlar informações relacionadas ao estoque.

Com o sistema é possível:

- ✅ Cadastrar fornecedores;
- ✅ Cadastrar produtos;
- ✅ Consultar produtos disponíveis;
- ✅ Visualizar quantidades armazenadas;
- ✅ Registrar entradas e saídas de produtos;
- ✅ Armazenar informações em banco de dados.

---

# 🚀 Tecnologias Utilizadas

## ☕ Java

Utilizado para desenvolvimento da lógica da aplicação, manipulação dos dados e integração com o banco de dados.

## 🖥️ Java Swing

Biblioteca utilizada para criação da interface gráfica do sistema.

Foi utilizada para desenvolver:

- Janelas;
- Botões;
- Campos de preenchimento;
- Tabelas;
- Mensagens de interação com usuário.

## 🗄️ MySQL

Banco de dados responsável pelo armazenamento das informações do sistema.

Principais informações armazenadas:

- Fornecedores;
- Produtos;
- Estoque;
- Movimentações.

## 🔌 JDBC

Responsável pela comunicação entre a aplicação Java e o banco MySQL.

Permite executar comandos SQL para:

- Inserção de dados;
- Consulta;
- Alteração;
- Exclusão.

## 📦 Maven

Utilizado para gerenciamento das dependências e organização do projeto.
